# Constants

The following constants are made available:

| Constant    | Contents                                                    |
|:------------|:------------------------------------------------------------|
| `ARGUMENTS` | Contains all CLI arguments, except for the executable path. |
| `SCRIPT`    | Contains the basename of the executable.                    |
